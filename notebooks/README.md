# Diretório de Notebooks (Experimentação)

Os notebooks originais utilizados para prototipação, exploração inicial de dados (EDA) e experimentação preliminar foram descontinuados nesta versão final do projeto.

Para garantir as melhores práticas de Engenharia de Software, evitar duplicidade de código e assegurar a reprodutibilidade estrita exigida (RQ-ER-01 e RQ-RP-03), todo o *pipeline* de processamento, treinamento, avaliação e auditoria visual foi refatorado para uma arquitetura modular de produção em Python puro.

O código-fonte oficial, orquestrado e pronto para execução encontra-se consolidado no diretório raiz e na pasta `/src`.
